# Household Net Worth Dashboard

A single-file personal finance dashboard to track household net worth across real estate, investments, cash, and liabilities.

## Features

- Real estate equity tracking (primary home + investment properties)
- Cash accounts across multiple banks (CAD + INR)
- Investment accounts: RRSP, TFSA, Non-Registered, Questrade
- Debt overview with mortgage and credit details
- CAD/USD/INR currency conversion
- Responsive dark-themed UI built with React + Tailwind CSS
